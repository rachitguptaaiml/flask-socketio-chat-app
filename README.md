# Real-time Chat Application using Flask-SocketIO

A simple real-time chat application built with Flask and SocketIO.

## Features

* Real-time messaging between clients
* Client-server architecture for efficient communication
* Easy to use and implement

## Setup/Run Instructions

1. Clone the repository: `git clone https://github.com/your-username/flask-socketio-chat-app.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Run the application: `python app.py`
4. Open your web browser and navigate to `http://localhost:5000`

## Sample Output

Client 1 said: Hello!
Client 2 said: Hi!
